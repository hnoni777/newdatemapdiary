개인정보처리방침

본 방침은 'HereWithYou' 앱(이하 '본 앱')의 개인정보 처리 방침을 설명합니다.

1. 수집하는 개인정보 항목

본 앱은 다음과 같은 정보를 처리합니다:

  • 위치 정보: 카드 생성 시 현재 위치 정보를 확인하기 위해 사용하며, 서버에 저장하지 않습니다.
  • 미디어/파일: 사용자가 선택한 사진을 카드로 꾸미기 위해 기기 내 미디어 접근 권한을 사용합니다.

2. 개인정보의 이용 목적

수집된 정보는 오직 다음의 목적으로만 이용됩니다:

  • 사용자의 위치 기반 데이트 기록 및 카드 생성 기능 제공
  • 생성된 카드의 기기 내 저장 및 공유 기능 제공

3. 개인정보의 제3자 제공

본 앱은 사용자의 동의 없이 개인정보를 외부에 제공하거나 공유하지 않습니다. 모든 데이터 처리는 기기 내에서 로컬로 원칙적으로 이루어집니다.

4. 개인정보의 보유 및 파기

본 앱은 서버를 운영하지 않으며 사용자의 어떠한 개인정보도 별도의 서버에 수집하거나 보유하지 않습니다. 모든 데이터는 사용자의 기기에만 존재하며, 앱 삭제 시 데이터는 함께 파기됩니다.

5. 사용자의 권리

사용자는 언제든지 앱의 권한 설정을 통해 정보 접근을 차단하거나 앱을 삭제함으로써 모든 데이터 처리를 중단할 수 있습니다.
